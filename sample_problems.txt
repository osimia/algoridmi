Задача 1. Квадратное уравнение
Решите уравнение: x² - 5x + 6 = 0

Решение:
Используем формулу корней квадратного уравнения.
D = b² - 4ac = 25 - 24 = 1
√D = 1
x₁ = (5 + 1)/2 = 3
x₂ = (5 - 1)/2 = 2

Ответ: x₁ = 3, x₂ = 2

Задача 2. Площадь треугольника
Найдите площадь треугольника со сторонами 3, 4, 5.

Решение:
Проверяем: 3² + 4² = 9 + 16 = 25 = 5²
Это прямоугольный треугольник.
S = (1/2) × 3 × 4 = 6

Ответ: 6

Задача 3. Арифметическая прогрессия
Найдите сумму первых 10 членов арифметической прогрессии: 2, 5, 8, 11, ...

Решение:
a₁ = 2 (первый член)
d = 3 (разность)
n = 10 (количество членов)

Найдем последний член:
a₁₀ = a₁ + (n-1)d = 2 + 9×3 = 2 + 27 = 29

Сумма:
S₁₀ = n(a₁ + a₁₀)/2 = 10(2 + 29)/2 = 10×31/2 = 155

Ответ: 155

Задача 4. Проценты
Товар стоил 1000 рублей. Цену увеличили на 20%, а затем снизили на 20%. 
Какова конечная цена товара?

Решение:
После увеличения: 1000 + 1000×0.2 = 1000 + 200 = 1200 руб
После снижения: 1200 - 1200×0.2 = 1200 - 240 = 960 руб

Ответ: 960 рублей

Задача 5. Система уравнений
Решите систему уравнений:
2x + y = 7
x - y = 2

Решение:
Из второго уравнения: x = y + 2
Подставим в первое: 2(y + 2) + y = 7
2y + 4 + y = 7
3y = 3
y = 1

Тогда x = 1 + 2 = 3

Проверка:
2×3 + 1 = 7 ✓
3 - 1 = 2 ✓

Ответ: x = 3, y = 1

Задача 6. Геометрическая прогрессия
Найдите пятый член геометрической прогрессии, если b₁ = 3 и q = 2.

Решение:
Формула n-го члена: bₙ = b₁ × qⁿ⁻¹
b₅ = 3 × 2⁴ = 3 × 16 = 48

Ответ: 48

Задача 10. Модуль числа
Решите уравнение: |x - 3| = 5

Решение:
Модуль раскрывается двумя способами:
1) x - 3 = 5  →  x = 8
2) x - 3 = -5  →  x = -2

Проверка:
|8 - 3| = |5| = 5 ✓
|-2 - 3| = |-5| = 5 ✓

Ответ: x₁ = 8, x₂ = -2
